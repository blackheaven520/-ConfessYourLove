HTML5动画 表白页面
支持移动端和电脑端

使用方法:
确保你有一个名为 romantic_music.mp3 的音乐文件放在与 HTML 文件相同的目录下，或者修改代码中 <audio> 标签的 src 属性为正确的路径。	
将上面的全部代码复制到一个纯文本编辑器中。	
保存文件为 .html 后缀（例如 mobile_romantic.html）。	
用浏览器打开，或者部署到 Web 服务器上，然后用电脑和手机分别访问进行测试。
